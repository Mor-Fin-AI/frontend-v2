# OpenClaw integration

OpenClaw is vendored in this repo:

```
mor-finance-user-dashboard/
├── openclaw/                    # git submodule — OpenClaw source
├── integrations/openclaw/       # MOR agent workspaces + config
│   ├── workspaces/mor-*/
│   ├── skills/mor-finance/
│   ├── openclaw.json            # generated by setup (gitignored)
│   └── state/                   # runtime (gitignored)
└── docs/agents/prompts/         # prompt source of truth
```

## Commands

```bash
npm run openclaw:setup      # sync prompts → workspaces, write openclaw.json
npm run openclaw:gateway    # start gateway on :18789
npm run dev:full            # MOR + OpenClaw together
```

See [integrations/openclaw/README.md](../../integrations/openclaw/README.md).

## Clone with submodule

```bash
git clone --recurse-submodules <repo-url>
# or after clone:
git submodule update --init --recursive
```
